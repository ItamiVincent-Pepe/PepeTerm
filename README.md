# 🐸 PepeTerm

**PepeTerm** — терминал для работы с сетевым оборудованием (коммутаторы, роутеры и прочее).  
Написан сисадмином для сисадминов. WPF / .NET 10 / Windows.

> Личный проект для практики C#. Бесплатно, открытый код, без лишнего.

## ✅ Что уже работает

- Подключение по **Telnet**
- Подключение по **SSH** (через SSH.NET)
- Вкладки на каждую сессию с кнопкой закрытия и **Ctrl+D**
- Боковая панель сохранённых хостов (двойной клик — автозаполнение)
- Тёмная тема (отдельный `DarkTheme.xaml`)
- Валидация ввода (хост, порт, протокол)

## 📦 Установка и запуск

### Требования

- Windows 10 / 11
- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- Visual Studio 2022

### Сборка из исходников

```bash
git clone https://github.com/yourusername/PepeTerm.git
cd PepeTerm
dotnet restore
dotnet build
dotnet run
```

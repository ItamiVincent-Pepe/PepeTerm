# 🐸 PepeTerm

**PepeTerm** — мощный эмулятор терминала для сетевого оборудования.  
Бесплатная open-source альтернатива MobaXterm и SecureCRT на WPF/.NET 10.

---

## 🚀 Возможности

- [x] Подключение по **Telnet** через plink.exe (PuTTY CLI)
- [x] Подключение по **SSH** через plink.exe
- [ ] Подключение по **Serial / COM-порт** (в разработке)
- [x] Вкладки с крестиком и кнопкой отключения
- [x] Боковая панель с деревом сохранённых хостов
- [x] Папки и вложенные папки
- [x] Drag & Drop для перемещения хостов между папками
- [x] Сохранение и загрузка хостов в JSON-файл
- [x] Иконки протоколов в дереве: 🖧 Telnet, 🔒 SSH
- [x] Иконка в системном трее (сворачивание вместо закрытия)
- [x] Сплеш-скрин при запуске
- [x] Тёмная тема в отдельном файле стилей
- [x] Чистая архитектура: Dialogs, Models, Services, Controls, Styles
- [x] Портативность: plink.exe в папке с программой (не требует установки)

## 📦 Установка

### Требования

- Windows 10/11
- .NET 10 SDK

### Сборка

```bash
git clone https://github.com/ItamiVincent-Pepe/PepeTerm.git
cd PepeTerm
dotnet restore
dotnet build
dotnet run
